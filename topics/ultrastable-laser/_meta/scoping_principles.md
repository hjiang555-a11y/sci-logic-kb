## 1. σ_y(τ=1 s) 主线原则（Primary-Metric Principle）

### 1.1 原则

> **超稳激光专题的唯一主线指标是 σ_y(τ=1 s)。**一篇论文是否值得 `breakthrough` 档，**首先**看它在 σ_y(1s) 维度上刷新了什么纪录；其他所有指标（线宽、频噪 PSD、相干时间、加速度灵敏度、长期漂移等）都是**次要或工程指标**，单独刷新不足以支撑 breakthrough。

动机：

- 超稳激光在时频计量链中的价值是**为光钟询问提供低 Dick 效应的询问激光**，1 s 量级与典型光钟询问循环直接耦合
- σ_y 的每一次台阶都驱动了下游光钟迭代；"亚赫兹 / mHz 线宽新纪录"历史上多次未带来下游性能提升
- 长期稳定性（hours–days）历来由**光频标自身 + 频率梳锁定链**解决（参见 `topics/frequency-standards/`），不是超稳激光的核心战场；超稳激光只需保证短期 σ_y 与漂移可控即可
- 把主线单一化，可以把档位判定、综合视图、记录榜、推理链条全部锚定到同一坐标轴上，避免"线宽 vs σ_y vs 相干时间"三维争议

### 1.2 档位判据（按主线单一化后的表）

| 信号 | 对 `contribution_type` 的影响 |
|------|------------------------------|
| **刷新全局 σ_y(1 s) 纪录**（ADEV/MDEV/OADEV/Hadamard 任一） | ✅ `breakthrough`（强信号） |
| **首次抵达某子分支 σ_y 热噪声极限**（光纤 FDL、SHB、空心光纤、新腔长等） | ✅ `breakthrough`（强信号） |
| **刷新某子分支（FP 腔 / 光纤干涉仪 / SHB）的 σ_y(1 s) 子 SOTA** | ✅ `breakthrough`（强信号） |
| **提出可复用的新稳频原理**（新 `pri.*`，且被后续论文引用） | ✅ `breakthrough` |
| 打破瞬时线宽纪录（sub-Hz / mHz），但 σ_y(1 s) 未刷新 | ⚠ `evidence`（单独不升档） |
| 打破长期漂移纪录 / hour-scale drift-free（σ_y @ ≥1000 s） | ⚠ `evidence`（长期稳定度由下游频标解决，本专题不升档） |
| 打破加速度灵敏度 / 振动灵敏度纪录，但 σ_y(1 s) 未刷新 | ⚠ `evidence`（工程指标） |
| 打破精细度 / 镀层损耗角 / Q 值纪录，但 σ_y(1 s) 未刷新 | ⚠ `evidence`（使能指标） |
| 工程紧凑化 / SWaP 压缩 / 可搬运 / 空间级，性能未跨越 regime | `evidence` |
| 换激光波长复用同一稳频方案 | `evidence` |

### 1.3 边界

- 本原则用于**档位判定**（`breakthrough` vs `evidence`），**不**改变节点建模的优先级——指标节点仍应如实填入线宽、频噪 PSD、加速度灵敏度等所有可量化项，只是不再因此升档
- "次要指标"在 YAML 中保留，但在 `INDEX_metrics.md` 和 `_meta/architecture.md` 的"Key Performance Records"中**不进入主线榜**
- 本原则**不**适用于其他专题：
  - 频率标准以**分数频率不确定度（accuracy）**优先
  - 光学频率梳需按 sub-topic 各自定义主线（见 TODO 登记）
  - 时间标尺与钟组以**长期时间保持**优先

### 1.4 "strong evidence" 边界案例

> 以下情况虽然仍是 `evidence`，但具有较强的建库复用价值，请在 YAML `meta.note` 中显式说明，以便未来快速检索：

- 打破次要指标纪录（如首次 sub-Hz 光纤激光线宽），即使 σ_y 未刷新
- 提供了跨 regime 的重要比对数据（如首次在 sub-5K 温度的 σ_y 测量）
- 给出了关键机制的工程化可行性验证（如首次商品化某原理）

### 1.5 关键指标换算原则

超稳激光论文常出现以下情况：报告了线宽 / 频噪 PSD，但**未报告 σ_y(1 s)**。处理规则：

- **优先从原文摘录 σ_y** — 哪怕是论文正文图中的一个数值点
- **次之从频噪 PSD 换算** — 若论文给出 S_y(f)，可用标准公式（Rubiola 手册 / IEEE 1139）换算到 σ_y(1 s)，并在 YAML note 中标注 `σ_y 由 PSD 换算，换算假设 ...`
- **避免从线宽反推 σ_y** — 线宽与 σ_y 的换算依赖噪声类型（白相 vs 白频 vs 闪变频率噪声），误差大，仅作为粗略估计并明确标注不确定性
- 若论文确实无法获得 σ_y 估值，在 YAML `meta.note` 中声明 `primary-metric missing: linewidth only` ——这本身不阻碍该论文入库（仍可归 `evidence`），但**不能据此升 breakthrough 档**

### 1.6 σ_y 类型标注规范（必须）

> 并非所有论文都明确标注 Allan 偏差的类型（普通 ADEV vs 修正 MDEV vs 重叠 OADEV vs Hadamard 方差）。不同类型对同一物理过程的数值不同（例如 MDEV 抑制白相噪声快于 ADEV），直接堆在一起比较会产生误导。

提取 `demonstrated_value` 时：

- **必须**在 `conditions` 或 value 字符串中注明 Allan 变体类型，例如：
  - `mod σ_y = 4×10⁻¹⁷`（明确为修正 Allan）
  - `σ_y = 3×10⁻¹⁶（ADEV）` 或 `σ_y = 3×10⁻¹⁶（overlapping ADEV）`
  - `σ_H = ...`（Hadamard）
- **若论文只写"Allan deviation"未明示变体**：在 YAML note 中标注 `Allan type: unspecified (assumed ADEV per convention)`，并在比较其他论文时不与已明确标注的 MDEV 数值直接并列排序
- **跨论文比较时**：在综合页面 / 记录榜里同一行尽量只列同类型（或明确标注类型差异），避免"世界纪录" overreach

### 1.7 "子分支 SOTA" 的界定

为避免把"某新颖实现路线的首次演示"误判为 evidence，本专题承认以下三个子分支各自的 σ_y SOTA 有独立的 breakthrough 地位：

1. **FP 腔分支**（主流，σ_y SOTA ~10⁻¹⁷）
2. **光纤干涉仪分支**（FDL / 自零差，σ_y SOTA ~10⁻¹⁵）
3. **光谱烧孔 SHB / 稀土晶体分支**（σ_y SOTA ~10⁻¹⁶）

> 若出现第四类全新稳频物理方案（如超辐射激光、布里渊激光达到超稳水平），可在本节追加第 4 类子分支 SOTA。
