## 二、专题间关系

时间频率计量的核心技术链：

```
超稳激光 ──→ 光学频率梳 ──→ 光钟
   │              │             │
   │              │             ├──→ 时间标尺
   │              │             └──→ 基础物理
   │              │
   │              └──→ 微波标准（通过梳齿分频）
   │
   └──→ 时间频率传递（远程比对需要相干光源）
```

**关键接口**：

| 接口 | 上游专题 | 下游专题 | 核心指标 |
|------|---------|---------|---------|
| 询问激光 → 光钟 | 超稳激光 | 光钟 | 激光线宽、相干时间 |
| 梳齿锁定 → 频率计数 | 超稳激光 + 光学频率梳 | 光钟比对 | 梳齿相位噪声 |
| 光纤传递 → 远程比对 | 超稳激光 | 时间频率传递 | 传输链路不稳定度 |
| 光-微波分频 → SI 秒 | 光学频率梳 | 微波标准 | 分频噪声贡献 |

## 四、建设优先级建议

### 第一梯队（与超稳激光耦合最紧密）

1. **光学频率梳**：超稳激光的直接下游；梳齿相位噪声直接影响光钟性能
2. **光钟**：超稳激光的主要应用场景；询问激光线宽是当前限制之一

### 第二梯队

3. **时间频率传递**：远程光钟比对的必要环节
4. **微波频率标准**：与光钟形成完整的"秒定义"链条

### 第三梯队

5. **时间标尺与钟组**：顶层应用
6. **基础物理应用**：前沿交叉领域

## 五、跨专题节点共享规则

### 5.1 通用原理

某些物理原理跨越多个专题，例如：
- Allan 偏差（表征方法）→ 所有专题
- 散粒噪声极限 → 超稳激光 + 光钟 + 频率梳
- Dick 效应 → 光钟 + 超稳激光

**处理方式**：若同一原理在不同专题中的公式形式、主导变量、工程含义明显不同，各专题保留独立 `pri.*` 节点（遵循跨分支原理隔离原则）。若确实是同一原理的同一表达，则在 `shared/` 目录中定义一次，各专题引用。

### 5.2 共享节点目录


## 六、演进原则

1. **渐进扩展**：保持一个主专题深耕，同时允许下游专题先以少量代表论文预热
2. **Schema 复用**：所有专题共用同一 SCHEMA.md 中的节点类型、关系类型和质量要求
3. **架构适配**：SCHEMA.md 中的"系统架构"部分按专题分区描述
4. **现有内容不破坏**：超稳激光的 78 篇论文 YAML 内容不变，仅调整存放路径
5. **向前兼容**：新专题的节点 ID 避免与已有专题冲突，通过前缀或命名空间区分
