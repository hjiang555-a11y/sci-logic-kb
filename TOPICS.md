## 专题间关系概览

时间频率计量的核心技术链：

```
超稳激光 ──→ 光学频率梳 ──→ 频率标准（光学+微波）
   │              │             │
   │              │             └──→ 时间标尺
   │              │
   │              └──→ 频率标准/微波（通过梳齿分频）
   │
   └──→ 时间频率传递（统一光学/微波链路；远程比对需要相干光源与链路补偿）

时频计量数学基础 ←── 所有专题共用（表征方法、噪声模型、方差体系）
```

详细接口定义与核心指标请参阅 SCHEMA.md。

## 跨专题节点共享

- 通用原理、方法、指标节点应放在 `topics/shared/` 目录
- 通过 `SHARED‑WITH` 关系引用，避免重复定义
- 共享节点的 ID 前缀为 `pri.`、`meth.`、`met.`，不带专题命名空间

## 如何使用

- 查看完整规范：请阅读 [SCHEMA.md](SCHEMA.md) 的「专题体系与演进」章节
- 添加新论文：进入相应专题目录，参照现有 YAML 模板
- 检查专题状态：运行 `python scripts/stats.py`
