### 如何获取 PDF 路径

```bash
WINDOWS_IP=$(ip route | grep default | awk '{print $3}')
# 获取论文的附件（PDF）信息
curl -s -H "Host: 127.0.0.1:23119" \
  "http://${WINDOWS_IP}:23119/api/users/19944378/items/{ZOTERO_KEY}/children" \
  | python3 -c "import json,sys; [print(i['data'].get('path','') or i['data'].get('filename','')) for i in json.load(sys.stdin) if i['data'].get('itemType')=='attachment']"
# PDF 在 WSL2 中的路径格式：/mnt/d/Users/hjian/Zotero/storage/{STORAGE_KEY}/{filename}.pdf
```
